**Moloch** is a powerful networking tool designed for **educational purposes** and **network stress testing**. This project aims to help developers and security researchers understand traffic patterns and test the resilience of their own infrastructure against high-load scenarios.

## 🚀 Features

* **Multi-Vector Support:** Support for TCP, UDP, and HTTP flood methods.
* **High Performance:** Optimized with multithreading for maximum packet throughput.
* **Customizable Parameters:** Easily adjust thread count, packet size, and target ports.
* **Lightweight:** Minimal dependencies, easy to deploy and run.
